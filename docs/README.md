## 구현해야하는 기능 목록

### 초기화
- 기본 메뉴 정보 관리
    - 각 메뉴는 메뉴 번호, 종류, 이름, 가격을 가진다.
    
- 기본 테이블 관리
    - 각 테이블은 번호를 가진다.
    
### 메인 화면
- 등록된 선택지의 번호만 입력해야한다.
    
### 주문 등록
- 테이블 목록을 보여준다.
    - 등록된 테이블 번호만 입력해야한다.
- 등록된 메뉴를 보여준다.
    - 등록된 메뉴 번호를 입력해야한다.
- 주문하기
    - 주문 수량은 1이상의 수여야한다.
    - 한 테이블에서 하나의 메뉴 당 99개까지만 주문할 수 있다.
        - 해당 메뉴 현재 수량 + 지금 주문 수량이 99개를 초과하지 않는지 검증

### 결제하기
- 테이블 목록을 보여준다.
    - 등록된 테이블 번호만 입력해야한다.
- 해당 테이블의 총 주문량을 보여준다.
- 결제하기
    - 등록된 선택지만 선택해야한다.
- 할인 적용 
    - 같은 종류의 메뉴 수량은 10개 단위로 10000원 씩 할인 된다.
    - 현금 결제는 5% 추가 할인 된다.
- 주문이 불가능한 경우 이유 출력 후 다시 메인 화면으로 이동한다.
- 최종 결제 금액 출력
